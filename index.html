Distance Estimator

Step-by-Step Guide

Setup Required

  • Cardboard tube - for sighting the object
  • Protractor - for measuring angles
  • 8-meter rope - for measuring distances
  • This app - for calculations (accessible via QR code)

Method 1 - Forward Step

Step 1: Sight the target object through the cardboard tube and measure its apparent size.
Step 2: Walk forward exactly 8 meters toward the object using the 8-meter rope as your guide.
Step 3: Sight the object through the tube again and measure the new apparent size.
Step 4: Enter both sizes plus the 8m distance into the app.

Method 2 - Parallax

Step 1: Sight the target object through the cardboard tube, noting its position.
Step 2: Move sideways exactly 8 meters (at a 90° angle to your original line of sight) using the rope.
Step 3: Measure the angular shift of the object using the protractor.
Step 4: Enter the angle measurement and 8m side distance into the app.

Result

The app calculates the actual distance to the object. You can use one method alone or combine both measurements for better accuracy.
Distance = 1 / (a₀ + a₁×forward_ratio + a₂×parallax_ratio)
Where coefficients are optimized through machine learning
